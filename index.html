Tentang Saya

Student at information systems engineering

Saya adalah Moh Hasan Efendi, seorang mahasiswa berusia 19 tahun yang tinggal di kota Kediri. Saat ini, saya sedang mengejar impian saya di Universitas Nusantara PGRI Kediri dalam bidang teknik sistem informasi. Saya memiliki minat khusus dalam bidang IT (Teknologi Informasi) dan tengah mengkhususkan diri dalam pengembangan sistem informasi. Selama perjalanan pendidikan saya, saya telah mengambil langkah berani untuk memahami berbagai aspek penting dalam dunia pemrograman. Saya memiliki pengetahuan mendalam dalam bahasa pemrograman seperti HTML, CSS, dan JavaScript, serta familiar dengan kerangka kerja CSS seperti Bootstrap dan Tailwind. Selain itu, komponen seperti Daisy UI dan jQuery di JavaScript juga menjadi bagian dari penguasaan saya. Namun, semangat saya dalam belajar tidak berhenti di sana. Saya memiliki keinginan yang kuat untuk mendalami JavaScript lebih mendalam, termasuk penggunaan library seperti React. Saya percaya bahwa dengan memahami dan menguasai alat-alat ini, saya dapat menjadi seorang front-end developer yang berkualitas dan mampu menciptakan antarmuka yang menarik dan responsif bagi pengguna. Selain itu, saya juga telah mulai menggali bahasa pemrograman lain seperti PHP dan kerangka kerja Laravel. Ini adalah bagian dari komitmen saya untuk menjadi seorang programmer yang komprehensif dan mampu menghadapi berbagai tantangan dalam pengembangan perangkat lunak. Dalam perjalanan saya sebagai seorang mahasiswa dan calon developer, saya berusaha untuk terus mengembangkan keterampilan dan pengetahuan saya dalam dunia programming. Saya percaya bahwa dengan dedikasi dan kerja keras, saya akan mencapai tujuan saya untuk menjadi seorang front-end developer yang berpengaruh dan sukses dalam industri teknologi.

Web CSS Grid

Batterai Detection (remake)

Proyek yang telah saya buat merupakan sebuah website yang merupakan hasil remake dari konten yang ada di YouTube. Proyek ini memungkinkan pengguna untuk memantau status sisa baterai dari suatu perangkat dengan menggunakan kombinasi HTML, CSS, dan JavaScript. Dalam proyek ini, terdapat elemen animasi yang menarik perhatian pengguna. Ketika perangkat sedang diisi atau dicharge, website akan secara dinamis menampilkan pesan "baterai charging" dan menyertakan animasi berbentuk baterai yang berwarna hijau. Ini memberikan indikasi visual yang jelas bahwa baterai sedang mengisi dan dalam kondisi baik. Sebaliknya, jika baterai perangkat telah habis atau dalam kondisi rendah, website akan menampilkan elemen animasi baterai yang berwarna merah. Ini memberikan pengguna peringatan visual yang berguna untuk mengingatkan bahwa perangkat memerlukan pengisian daya segera.
